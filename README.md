# Materially
### Material 3/You theme for Jekyll

#### **_WIP_**

## Setup/Install

Add this line to your Jekyll site's `Gemfile`:

```ruby
gem "materially-jekyll"
```

And add this line to your Jekyll site's `_config.yml`:

```yaml
theme: materially-jekyll
```

And then execute:
```bash
bundle
```
